This script tells you the current playing state of a running Windows Media Player instance. I put this script here because of how complex WMP makes it to tell you its state. I hope this might help somebody.

Resources used:
	* The "remoteHost" example of https://github.com/Microsoft/Windows-classic-samples
	* manuell's post here: https://stackoverflow.com/a/19571308
	
Tested with Windows 10 1703 / WMP 12 / AutoHotkey x64 1.1.26.01
